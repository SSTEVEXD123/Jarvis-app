# Jarvis Personal Assistant

## Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/SSTEVEXD123/Jarvis-app.git
   ```
2. Navigate to the project directory:
   ```bash
   cd Jarvis-app
   ```
3. Install the required dependencies:
   ```bash
   npm install
   ```

## Usage
To start the application:
```bash
npm start
```

Follow the on-screen instructions to interact with your personal assistant.
